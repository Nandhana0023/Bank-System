# Bank Management System
Author: Nandhana MA

Description: 
- This is a simple Bank Management System developed using OOP in Java.
- It allows users to perform basic banking operations such as depositing money, withdrawing money, checking balance, and viewing account details through a menu-driven console interface.
##  Technologies Used

- Java

- VScode

- OOPs Concept
## Features

- Create a bank account with account holder name, account number, and initial balance

- Deposit money to the account

- Withdraw money with balance validation

- Display account details

- Check account balance
## How it works
1. The program creates a BankAccount object with default values.

2. A loop-based menu allows the user to:

- Add or withdraw money

- View details

- Exit the system
